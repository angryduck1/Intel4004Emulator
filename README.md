# Intel 4004 Emulator

## Simple Intel 4004 Emulator

1. **Set the path** to your ASM code in `main.cpp`.
2. **Compile** `main.cpp`.

### Implemented Instructions

| **Instruction** |
|-----------------|
| **NOP**         |
| **LDM**         |
| **ADD**         |
| **SUB**         |
| **INC**         |
| **CMC**         |
| **XCH**         |
| **JCN**         |

### Features

- All 15 registers
- Accumulator
- Carry flag
